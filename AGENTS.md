# AGENTS.md - O2M Design

O2M Design is a calm editorial and portfolio site focused on clarity, systems, continuity, and usable complexity.

## Publishing posture

Preserve O2M /thinking as a calm editorial continuity surface.

Keep article publishing markdown-driven and low-complexity.

Preserve the existing design patterns unless explicitly instructed otherwise.

## Scope guardrails

Do not introduce CMS behavior, search, comments, personalization, AI overlays, engagement mechanics, backend services, dashboards, or major publishing workflow complexity without explicit approval.

Do not change the visual design system or article interaction model unless specifically approved.

## Validation

Run build checks before reporting completion:

```bash
npm run build
```

Report changed files and validation status after each publishing task.

## Deployment

Do not manually deploy without Jorge approval.
